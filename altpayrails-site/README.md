# AltPay Rails

Interactive multi-rail payment checkout system for high-risk merchants.

**Live Demo:** https://altpayrails.com

## Features

- **7 Payment Rails:** BTCPay Server, Cash App, Monzo, Skrill, Zelle, Card-to-Crypto, Credit Card
- **5-Second Polling:** Real-time payment verification with "Did You Know" facts
- **P2P Support:** Expandable cards with copy buttons, QR codes, and deep links
- **3D Credit Card:** Interactive card preview with flip animation
- **WebGL Background:** Animated fluid background

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- Framer Motion
- Three.js / React Three Fiber
- GSAP
